Glowing Button

You can change the color of your choice here. (linear-gradient(#4d00ff, #673fd7))

.mediva-glow-btn {
    background-image:  linear-gradient(#4d00ff, #673fd7);
    border-radius:  32px !important;
    transition:  transform 0.3s ease-in-out, box-shadow 0.3s ease-in-out;
    background-color:  #00000000 !important;
    backdrop-filter:  blur(20px);
    box-shadow:  0 -7px 20px rgba(110, 48, 255, .27),
        1px 7px 20px 1px rgba(103, 63, 215, .65),
        inset 6px -6px 16px rgba(255, 255, 255, .08),
        inset 0 0 20px rgba(255, 255, 255, .08);
    font-family:  "Poppins-medium";
}

How to implement:

1. Copy the CSS above and paste it on the Custom CSS.
2. Remove the Box Shadow option in Advanced settings.
3. Add the "mediva-glow-btn" in button Custom Class.

Glowing Text

You can change the hex colors anytime.

.mediva-glow-txt {
    color:  #ffffff; /* white */
    text-shadow:  0 0 5px #00e0ff,
        0 0 10px #00e0ff,
        0 0 20px #00e0ff,
        0 0 40px #00e0ff !important;
}

How to implement:

1. Copy the CSS above and paste it on the Custom CSS.
2. Add the "mediva-glow-txt" in headline, subheadline or paragraph Custom Class.

Nice FAQ

You can change the hex colors anytime.

.faq-separated-child {
    border-radius:  40px !important;
    padding:  12px;
    background-color:  rgba(255, 255, 255, .03);
    background-image:  linear-gradient(rgba(255, 255, 255, 0),
        rgba(255, 255, 255, .05));
    line-height:  40px !important;
    border:  1px solid rgba(255, 255, 255, .10) !important;
}

How to implement:

1. Copy the CSS above and paste it on the Custom CSS.
2. Adjust the FAQ Color Options on Advanced settings.
  • Title text color (open) = #ffffff or white
  • Title text color (closed) = #ffffff or white
  • Title background color (open) = transparent
  • Content text color = #ffffff or white
  • Content background color = transparent
